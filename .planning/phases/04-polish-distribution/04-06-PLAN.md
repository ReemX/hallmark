  <what-built>
    Plans 04-01 through 04-05 wired the entire updater plumbing (background-check, modal, install command), but `tauri.conf.json` `plugins.updater.pubkey` is the literal string `PLACEHOLDER_REPLACE_AT_RELEASE`. Without a real keypair, the CI workflow's signing step will fail and the runtime updater check will fail signature verification (correctly — the plugin refuses to install an unsigned/unverified binary).

    This plan cannot AUTOMATE keypair generation because:
    1. `tauri signer generate` writes the PRIVATE key to a local file. The private key MUST be uploaded as a GitHub secret AND the local copy DELETED — both actions require human discretion.
    2. Setting GitHub secrets requires the maintainer's GitHub login session.
    3. There's no CLI/API path that lets Claude perform "generate key, paste into secret, delete file, commit pubkey" without the private key passing through a context-window or a local FS copy that persists past the keypair generation step.

    Per CONTEXT.md D-21: "Private key pasted directly into TAURI_SIGNING_PRIVATE_KEY GitHub Actions secret + TAURI_SIGNING_PRIVATE_KEY_PASSWORD if password-protected. Local copy DELETED immediately after upload."
  </what-built>
  <how-to-verify>
    Execute the following steps manually. Each step is a hard requirement.

    **Step 1: Generate the keypair.** In a terminal where Tauri CLI 2.x is available:
    ```sh
    cargo tauri signer generate -w hallmark-private.key
    ```
    OR (on Windows PowerShell with `tauri-cli` installed):
    ```powershell
    cargo tauri signer generate -w hallmark-private.key
    ```

    https://github.com/ReemX/hallmark/settings/secrets/actions

    Click "New repository secret" twice:
    - Name: `TAURI_SIGNING_PRIVATE_KEY`
      Value: (paste the FULL contents of `hallmark-private.key`, including any header lines)
    - Name: `TAURI_SIGNING_PRIVATE_KEY_PASSWORD`

    ```yaml
              GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

              $version = $env:GITHUB_REF_NAME -replace '^v', ''
              $exe = "src-tauri/target/release/hallmark.exe"
    ```
