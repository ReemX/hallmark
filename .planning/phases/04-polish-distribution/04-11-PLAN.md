<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WebView→OS shell | tauri-plugin-shell::open invokes Windows ShellExecute / equivalent — passes the URL string to the OS-level URL handler |
| Capability allowlist | Tauri's capability resolver gates which URLs the WebView can pass to shell.open — the security boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
